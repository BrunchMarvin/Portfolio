# Public Icons Folder

Add SVG icons, favicon variants, and icon assets here.

## Suggested Contents:
- `twitter.svg` - Social media icons
- `linkedin.svg`
- `github.svg`
- `email.svg`
- `phone.svg`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png`

Icons can be used as:
```html
<img src="/icons/twitter.svg" alt="Twitter" />
```

Or imported in React components.
