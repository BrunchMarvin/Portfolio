# Public Images Folder

Add portfolio project images, headshots, and other static images here.

## Suggested Contents:
- `profile.jpg` - Your professional headshot
- `project-1.jpg` - Portfolio project images
- `project-2.jpg`
- `testimonial-1.jpg` - Client/testimonial images

Images in this folder can be referenced in your components as:
```html
<img src="/editing-portfolio/images/profile.jpg" alt="Description" />
```

Or without the repo name if using relative paths:
```html
<img src="images/profile.jpg" alt="Description" />
```
